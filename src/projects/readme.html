Coder Academy JavaScript Hackathon - Article Reading Service.

By Shelby El-rassi & Benjamin Rochlin

Deployed Site GitHub
  • JavaScript
  • Amazon Polly
  • BootStrap
  • Cheerio

As a team we created this application as part of a JavaScript Hackathon at Coder Academy. The theme was open so we created a article reading service application with a custom web-scraper using Cheerio and Amazon Polly for the voice synthesising and URL output.

The purpose of the application is an instant audio listening output for news/medium/blog articles which can avoid ads, unnecessary text and it also can be used as a great accessibility tool.

Future features include an email input for file sending, outputting an mp4 audio file, custom voice settings and fine tuning the web-scraper to be compatible with a larger range of news websites.
