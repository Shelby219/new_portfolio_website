Taskio.

Coder Academy Express Hackathon - Gamified Task Tracker.

By Shelby El-rassi & Alexander Qiang

Deployed Site GitHub
  • Express
  • MongoDB
  • Node JS
  • Pug JS

Purpose: The app is made to simplify our daily tasks schedule and build good habits. Everyday Task Tracker with a gamified twist. When the need for keeping task lists is present, with the fun of earning points is extra incentive for task completion.

As a team we built an app whereby a user can register an account and add details and avatar. Add new tasks, edit and delete, then once marked complete points are awarded. The user dashboard is the core app interface.
