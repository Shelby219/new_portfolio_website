TLDR; A recipe search applicaition which users can search using ingredients in their fridge/pantry along with preferences.

The purpose of this application is to generate recipes based on ingredients the in the users fridge and pantry. The idea stemmed from the first Covid lockdown in which stores sold out of a lot of favourite grocery items of customers. The application has benefits including keeping to a budget, recipe inspiration, minimise food waste, and search based on dietary needs and intolerances.

As a team we built this application and currently its features include, User account creation and authorisation, adding/deleting new fridge and pantry ingredients, selecting preferences for filtering (ie. Gluten-Free), browsing recipes based off these parameters, and saving recipes for later use.

Future features that would further improve this application include, Oauth sign up/login, search history capture, recipe collection saving, categories for fridge and pantry ingredients and a large filter feature for the search page to filter recipes returned.
